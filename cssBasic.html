  6. 4 types of CSS selectors

    • The element selector

      • It selects element name.

    • The class selector

      • On html, use the class attribute: <tag class="className">
      in <style>, .className{ }
      class ="className1 className2"
      The class selector selects elements with a specific class attribute. The last class selector has a priority of the same class selectors.

    • The id selector

      • On html, use the id attribute: <tag id="idName">
      in <style>, #idName{ }
      id selector has the first priority of all the selectors.
      The id value have to be unique within a page.
      ex) id card, identify: indicate who/what is

    • complexed selectors

      • These selector can be mixed on a tag.
      ex) <tag class="classname" id="idname">

    • Grouping selectors

      • To group selectors for minimizing the code, seperate each selector with a comma.
      ex) h1, h2, p { }

    • Color Names

      • Tomato

      Orange

      Gray

      SlateBlue

      LightGray
    • property: background-color
      • value: color names

    • property: color (for text)
      • value: color names

    • property: border(for border color)
      • value: 2px solid colorNames

      Color value

      • RGB Value

        • rgb(red, green, blue)

      • HEX value

        • #rrggbb

      • HSL Value

        • hsl(hue, saturation, lightness)

        - Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
        - Saturation is a percentage value, 0% means a shade of gray, and 100% is the full color.
        - Lightness is also a percentage, 0% is black, 50% is neither light or dark, 100% is white

      • RGBA Value

        • rgba(red, green, blue, alpha)

      • HSLA Value

        • hsla(hue, saturation, lightness, alpha)

    B. font

    • two types of font family name
      • generic family - base fonts(serif, sans-serif,monospace)
      font family - a specific font family

    • property: font-family
      • fallback system: If the browser does not support the first font , it tries the next font.
      Start with the font you want, and end with a generic family
      value: "Times New Roman", Times, serif;

    • property: font-style
      • value: normal; italic; oblique;

    • property: font-size
      • - absolute size
      - Relative size
      * default font size: 16px = 1em;
      value: px; em;(The em size unit is recommended by the W3C.)

      -Responsive Font Size
      value: vw;(viewpoint width)
      Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.

    • property: font-weight
      • value: normal; bold;
      Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.
